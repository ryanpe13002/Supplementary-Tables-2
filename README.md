This repository contains supplementary tables from the publication "Intron retention regulates STAT2 function and predicts immunotherapy response in lung cancer" by Englander *et al.*

Files included in this repository:

- Supplementary Table 1: Patient and LR-seq library metadata
- Supplementary Table 2: Characteristics and expression of long-read derived isoforms detected in lung tumors
- Supplementary Table 3: Coding and UTR predictions of long-read-derived isoforms detected in lung tumors
- Supplementary Table 4: Long-read derived isoforms in interferon-related genes
- Supplementary Table 5: Survival and differential gene expression analysis from OAK, TCGA, and IMvigor210
- Supplementary Table 6: Reagent information
